Desarrollo Web

En este curso aprenderás a crear tu sitio web con en las mejores prácticas del desarrollo web, trabajando con HTML y CSS. Con herramientas para optimizar tu sitio web, implementando GIT, y preprocesadores como SASS. Al finalizar, sabrás cómo aplicar Bootstrap a tus proyectos y el uso del SEO en tus desarrollos.

JavaScript

En este curso aprenderás los fundamentos del lenguaje de programación más usado en la actualidad, con el cual es posible crear aplicaciones de todo tipo. Reconocerás la utilidad de las librerías estudiando jQuery, y cómo aplicar técnicas de desarrollo para apps modernas con AJAX.

React Js

En este curso aprenderás a programar por componentes, mediante Javascript, JS, ES6, y también conocerás las ventajas de la utilización del flujos de datos. Asimismo, aplicarás el manejo de rutas utilizando Firebase, y comprenderás la utilización del virtual DOM mediante los desarrollos de React JS.

Python

En este curso aprenderás las bases de programación uno de los lenguajes más populares en estos tiempos. Partirás desde sus fundamentos, para luego abarcar módulos y sintaxis, hasta el uso de reglas para crear tus primeras aplicaciones. Desarrollarás una app web estilo blog de la mano de Python en Django Framework.

SQL

En este curso aprenderás las nociones centrales de las bases de datos relacionales. Crearás una base de datos relacional desde cero hasta la inserción de la información, e implementación de procesos de automatización para el mantenimiento de la base. Implementarás consultas SQL avanzadas para generar reportes.

C#

En este curso aprenderás los conceptos y la sintaxis del lenguaje desarrollado por Microsoft C#. A través del Framework .NET aprenderás el circuito completo que debe hacer un programador para poder llevar a cabo el back end de una Aplicación Web.
